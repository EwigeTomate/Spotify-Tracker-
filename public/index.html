Spotify Developer API Setup

Dieser Tracker läuft lokal auf deinem Rechner. Um deine Spotify-Spieldaten abzufragen, musst du deine eigene Client ID und dein Secret hinterlegen.

1
App registrieren

Öffne das Spotify Developer Dashboard, logge dich ein und klicke auf "Create app".

2
Redirect URI eintragen

Trage in den App-Einstellungen unter Redirect URIs die folgende URL ein:

http://127.0.0.1:3000/api/auth/callback
3. Spotify Account verbinden

Klicke unten, um dich bei Spotify einzuloggen und das automatische Musik-Tracking freizugeben.
